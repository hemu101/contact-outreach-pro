## 🎯 Three Simple Steps

### Step 1: Get Supabase Credentials (5 min)
```
1. Go to app.supabase.com
2. Create new project
3. Copy URL and Anon Key from Settings > API
```

### Step 2: Configure App (1 min)
```bash
echo 'VITE_SUPABASE_URL=your-url
VITE_SUPABASE_ANON_KEY=your-key' > .env.local
```

### Step 3: Setup Database & Run (3 min)
```bash
npm install
# In Supabase: Copy scripts/database-migration.sql and run in SQL Editor
npm run dev
# Visit http://localhost:8081
```

**Done! Your app is running.** 🎉
